# <img src="airootfs/etc/calamares/branding/hujmanos/logo.png" width="50" align="center"> HujmanOS

**HujmanOS** is a specialized, high-performance Linux distribution based on Arch Linux. Engineered for **Cybersecurity Professionals**, **Penetration Testers**, and power users, it blends the cutting-edge aesthetics of the **Hyprland** compositor with a massive arsenal of security and forensics tools.

## üöÄ Key Features

* **Modern Tiling UI:** Powered by **Hyprland** (Wayland) with custom-preconfigured Dotfiles for maximum workflow efficiency.
* **Offensive Security:** Built-in support for advanced vulnerability research, specifically targeting **XSS** and **SQL Injection**.
* **Windows Compatibility:** Integrated **Wine**, **Winetricks**, and **Bottles** to run Windows-based security tools and software seamlessly.
* **Professional Productivity:** Pre-installed with **LibreOffice**, **Chromium**, **VLC**, and **GIMP** for daily operations.
* **Seamless Installation:** Features the **Calamares** graphical installer with a custom HujmanOS branding experience.

## üíø Installation

1.  Download the latest ISO from the [Releases] section.
2.  Flash the ISO to a USB drive using **Ventoy** or **Etcher**.
3.  Boot from the USB and select "Launch HujmanOS".
4.  Run the **Calamares** installer from the menu to perform a permanent install.
